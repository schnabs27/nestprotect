# NestProtect

A disaster preparedness web application that helps users prepare for, respond to, and recover from natural disasters. Provides hazard-specific checklists, resources, and risk assessments based on ZIP code location.

## Tech Stack

- React 18 + TypeScript + Vite
- Tailwind CSS + shadcn/ui
- React Router v6
- TanStack Query v5
- Supabase (auth + database)

## Running Locally

```sh
npm install --legacy-peer-deps
npm run dev        # http://localhost:8080
npm run build      # production build
```

## Deployment

Hosted on Vercel. Auto-deploys on push to the `main` branch.
